# สถานะงาน: ชีทเคมีทั่วไปตามแนวทาง สสวท.

อัปเดต: 20 กรกฎาคม 2569

## พร้อมให้ตรวจ

- บทที่ 1: ความปลอดภัยและทักษะในปฏิบัติการเคมี
  - หน้าเว็บและลิงก์ดาวน์โหลด PDF พร้อมใช้งาน
  - ชีทฉบับปัจจุบันมี 20 ข้อ และผ่านผู้ตรวจอิสระ `issues = 0`
  - เป็น **draft** รอเจ้าของ (แม่นเคมี) รีวิว

## ค้างก่อนเริ่มบทที่ 2

บทที่ 1 ต้องขยายก่อนให้เป็นมาตรฐานสำหรับบททั่วไปทั้งหมด:

1. ขยายแบบฝึกจาก 20 ข้อเป็นอย่างน้อย 60 ข้อ โดยกระจายทุกผลการเรียนรู้
2. เพิ่มเนื้อหาและแบบฝึกต้นฉบับเรื่อง NFPA, การอ่าน SDS จากสถานการณ์, การคัดแยกของเสีย,
   การเลือกเครื่องชั่ง, การออกแบบการทดลอง และจริยธรรมทางวิทยาศาสตร์
3. ตรวจอิสระแบบคำนวณใหม่ทุกตัวเลข → แก้ → ตรวจซ้ำจน `issues = 0`
4. สร้าง PDF ใหม่ ตรวจภาพทุกหน้า และทดสอบหน้าเว็บจริง

## มาตรฐานสำหรับบทต่อไป

- ทำทีละบทและให้เจ้าของตรวจรับก่อนข้ามบท
- เนื้อหาต้องครอบคลุมผลการเรียนรู้ของ สสวท. สำหรับบทนั้น
- แบบฝึกอย่างน้อย 60 ข้อ (เพิ่มเมื่อความกว้างของบทต้องการ) และไม่เติมจำนวนด้วยข้อซ้ำ
- ทุกข้อเป็นโจทย์แต่งใหม่, ผ่าน validator, ผู้ตรวจอิสระ และการตรวจตัวเลขด้วย `node -e`
- ทุกฉบับเป็น draft จนกว่าเจ้าของจะอนุมัติ
