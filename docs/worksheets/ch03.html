ตารางธาตุและสมบัติของธาตุ

หมวดนี้คือ "หมวดทำคะแนน" ของ สอวน. ค่าย 1 — ข้อสอบชอบออก 2 แนวคือ (1) ให้เรียงลำดับสมบัติ (ขนาด, IE, EN) แล้วดูว่าเราเข้าใจ เหตุผล เบื้องหลังแนวโน้มจริงไหม และ (2) ให้ข้อมูลปริศนา เช่น ค่า IE หลายค่า แล้วให้ทายว่าเป็นธาตุหมู่อะไร ถ้าจำแค่แนวโน้มแบบท่องนก จะโดนข้อยกเว้นดักทันที ดังนั้นต้องแม่นทั้งแนวโน้มหลัก + ข้อยกเว้น + เหตุผลระดับโครงสร้างอิเล็กตรอน

1. การจัดตารางธาตุ: หมู่ คาบ และบล็อก s p d f

ตารางธาตุยุคปัจจุบันเรียงตาม เลขอะตอม (ZZZ = จำนวนโปรตอน) ไม่ใช่มวลอะตอม (นี่คือจุดที่ Moseley แก้ปัญหาของ Mendeleev) หลักการอ่านตาราง:

  • คาบ (Period) = แถวแนวนอน บอก "จำนวนระดับพลังงานหลัก" ที่มีอิเล็กตรอน → คาบ = ค่า nnn สูงสุดของอิเล็กตรอน
  • หมู่ (Group) = แถวแนวตั้ง ธาตุหมู่เดียวกันมี เวเลนซ์อิเล็กตรอนเท่ากัน → สมบัติเคมีคล้ายกัน
  • บล็อก = ดูว่าอิเล็กตรอนตัวสุดท้ายเติมลงออร์บิทัลชนิดไหน
บล็อก อิเล็กตรอนสุดท้ายลงที่ ตำแหน่งในตาราง ตัวอย่าง
s ออร์บิทัล sss หมู่ IA, IIA (+ He) Na, Ca
p ออร์บิทัล ppp หมู่ IIIA–VIIIA Cl, Ne
d ออร์บิทัล ddd ธาตุแทรนซิชัน (หมู่ B) Fe, Cu
f ออร์บิทัล fff แลนทาไนด์/แอกทิไนด์ Ce, U

เทคนิคหาหมู่/คาบจากการจัดเรียงอิเล็กตรอน: เขียน configuration ก่อนเสมอ แล้ว

  • คาบ = nnn สูงสุด
  • หมู่ A = จำนวนอิเล็กตรอนในชั้นนอกสุด (เฉพาะ sss + ppp ของ nnn สูงสุด)

2. ขนาดอะตอมและขนาดไอออน

ขนาดอะตอมถูกกำหนดโดยการชักเย่อระหว่าง 2 แรง:

  1. แรงดึงจากนิวเคลียส — ยิ่งประจุนิวเคลียสสุทธิ (effective nuclear charge, ZeffZ_{eff}Zeff​) มาก อิเล็กตรอนยิ่งถูกดึงเข้า → เล็กลง
  2. จำนวนระดับพลังงาน — ยิ่ง nnn มาก อิเล็กตรอนนอกสุดยิ่งอยู่ไกล → ใหญ่ขึ้น

โดยประมาณ Zeff=Z−SZ_{eff} = Z - SZeff​=Z−S เมื่อ SSS คือจำนวนอิเล็กตรอนชั้นใน (shielding)

แนวโน้ม:

  • ในคาบเดียวกัน (ซ้าย → ขวา): nnn เท่าเดิม แต่ ZZZ เพิ่ม อิเล็กตรอนชั้นในเท่าเดิม → ZeffZ_{eff}Zeff​ เพิ่ม → ขนาดเล็กลง
  • ในหมู่เดียวกัน (บน → ล่าง): nnn เพิ่ม ชั้นอิเล็กตรอนเพิ่ม → ขนาดใหญ่ขึ้น (ชนะผลของ ZZZ ที่เพิ่ม)

ขนาดไอออน — 3 กฎที่ต้องจำ:

  1. ไอออนบวกเล็กกว่าอะตอมเดิมเสมอ (Na+<\text{Na}^+ <Na+< Na) เพราะเสียชั้นนอก + แรงดึงต่ออิเล็กตรอนที่เหลือมากขึ้น
  2. ไอออนลบใหญ่กว่าอะตอมเดิมเสมอ (Cl−>\text{Cl}^- >Cl−> Cl) เพราะอิเล็กตรอนเพิ่มแต่โปรตอนเท่าเดิม แรงผลักระหว่างอิเล็กตรอนเพิ่ม
  3. ไอโซอิเล็กทรอนิก (อิเล็กตรอนเท่ากัน): ตัวไหน ZZZ มาก ตัวนั้นเล็ก — เพราะโปรตอนเยอะกว่าดึงอิเล็กตรอนชุดเดียวกันแรงกว่า

3. พลังงานไอออไนเซชัน (Ionization Energy, IE)

นิยาม: พลังงานน้อยที่สุดที่ใช้ดึงอิเล็กตรอน 1 ตัวออกจากอะตอม (หรือไอออน) ใน สถานะแก๊ส

แนวโน้ม: ตรงข้ามกับขนาดอะตอม — ในคาบ ซ้าย→ขวา IE เพิ่ม (อะตอมเล็ก ดึงแน่น), ในหมู่ บน→ล่าง IE ลด (อะตอมใหญ่ อิเล็กตรอนนอกสุดอยู่ไกล)

4. สัมพรรคภาพอิเล็กตรอน (Electron Affinity, EA)

นิยาม: พลังงานที่เปลี่ยนแปลงเมื่ออะตอมในสถานะแก๊สรับอิเล็กตรอน 1 ตัว

X(g)+e−→X−(g)\text{X}(g) + e^- \to \text{X}^-(g)X(g)+e−→X−(g)

ธาตุส่วนใหญ่ คายพลังงาน เมื่อรับอิเล็กตรอน (EA เป็นลบตามอนุสัญญาเทอร์โมไดนามิกส์) ยิ่งคายมาก = ยิ่ง "อยากได้" อิเล็กตรอน

แนวโน้ม: คล้าย IE คือ ซ้าย→ขวา ชอบรับอิเล็กตรอนมากขึ้น, บน→ล่าง ลดลง — แต่มีข้อยกเว้นสำคัญมาก:

  • Cl คายพลังงานมากกว่า F (−349-349−349 เทียบกับ −328-328−328 kJ/mol) เพราะ F อะตอมเล็กมาก อิเล็กตรอนแน่น รับตัวใหม่เข้าไปแล้วเกิดแรงผลักสูง → หมู่ VIIA ตัวที่ EA เด่นสุดคือ Cl ไม่ใช่ F
  • หมู่ IIA และ VIIIA แทบไม่รับอิเล็กตรอน (บางค่าเป็นบวก) เพราะ configuration เดิมเสถียรอยู่แล้ว (ns2ns^2ns2 เต็ม / ครบออกเตต)

ข้อสอบมักถามเชิงเปรียบเทียบ เช่น "ธาตุใดรับอิเล็กตรอนแล้วคายพลังงานมากที่สุด" — คำตอบในตัวเลือกที่มี F กับ Cl ให้ตอบ Cl

5. อิเล็กโทรเนกาติวิตี (Electronegativity, EN)

นิยาม: ความสามารถของอะตอมในการ ดึงอิเล็กตรอนคู่ร่วมพันธะ เข้าหาตัวเอง — ต่างจาก EA ตรงที่ EN ใช้กับอะตอมที่ "อยู่ในพันธะแล้ว" และไม่มีหน่วย (สเกลพอลิง)

แนวโน้ม: ซ้าย→ขวา เพิ่ม, บน→ล่าง ลด → F มากสุด (4.0) และมุมล่างซ้าย (Cs, Fr) น้อยสุด (~0.7) — แก๊สมีสกุลส่วนใหญ่ไม่กำหนดค่าเพราะแทบไม่เกิดพันธะ
