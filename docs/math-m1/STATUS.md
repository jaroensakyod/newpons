# สถานะงาน: คณิตศาสตร์ ม.1

อัปเดต: 20 กรกฎาคม 2569

## บทที่กำลังทำ

**ทศนิยมและเศษส่วน** ตามหนังสือรายวิชาพื้นฐานคณิตศาสตร์ ม.1 เล่ม 1 ของ สสวท.

ขอบเขต:

1. ค่าประจำหลัก ค่าสัมบูรณ์ และการเปรียบเทียบทศนิยม
2. การบวก ลบ คูณ หารทศนิยม และโจทย์ปัญหา
3. ความหมาย การเปรียบเทียบ และการดำเนินการของเศษส่วน
4. ความสัมพันธ์ระหว่างทศนิยมและเศษส่วน
